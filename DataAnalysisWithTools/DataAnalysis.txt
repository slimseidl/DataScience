-   Not Linear

    - Ask Questions
        - problem to solve?
        - trends?

    - Wrangle
        - Gather Data 
        - Assess data 
            - quality / structure
        - Clean Data 
            - modify, replace values, remove outliers, etc
    - Exploratory Data Analysis (EDA)
        - Patterns
        - Visuals 
        - Feature Engineering
    - Conclusions
        - predictions / decisions 
        - ML 
    - Communicate Findings
        - Share
        - Make decisions
